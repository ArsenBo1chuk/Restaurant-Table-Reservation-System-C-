# Restaurant Table Reservation System (C)

A console-based application written in C for managing restaurant table reservations.

## Features
- Add new table reservations
- Prevent booking for past dates
- Prevent double booking of the same table at the same time
- Search reservations by customer name
- Search reservations by date
- Edit existing reservations
- Delete reservations
- Display all reservations in a formatted view

## Technologies
- C language
- Standard C libraries
- Time and date validation using `<time.h>`

## Project Purpose
This project was created as a learning exercise to practice:
- Structures (`struct`)
- Arrays
- Functions
- String manipulation
- Input validation
- Basic date handling in C

## How to Run
1. Compile the program:
```bash
gcc main.c -o reservation
```
## Run the program
```bash
./reservation
```

## Notes
- Maximum number of reservations: 20
- Console-based application
- Designed for educational purposes
